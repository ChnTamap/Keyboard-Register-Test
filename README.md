## 串口方案（FIFO）

先发低电平再发高电平，低电平是上个位的，高电平是当前位的。由于串口固定了下降沿（停止位+起始位）必须等周期，因此一个字节必须发超过一个位才能有机会规避这个问题

主频:
- 80MHz : 10MHz : 200ns 300ns 400ns
- 60MHz : 7.5MHz : 200ns 266ns 400ns
- 48MHz : 6MHz : 333ns
- 36.9MHz : 4.6125MHz : 216.8ns
